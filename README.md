# API: LabenuSystem

### Descrição
- API para o sistema de uma escola
- Principais funcionalidades: cadastrar/visualizar turmas, docentes e estudantes.

### Tecnologias utulizadas:
- Typescript
- Node.js
- Dotenv
- Express
- MySQL
- Knex
